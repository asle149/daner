# 다너 (Daner)

> 단어로 모이는 작은 커뮤니티

## 다너가 뭔가요

단어 하나가 하나의 방이 됩니다.

"퇴근"을 검색하면, 같은 단어를 떠올린 사람들이 모여 한마디씩 남긴 방이 열려요. 처음 도착한 단어라면, 한마디를 남기는 순간 그 단어의 방이 생겨납니다.

게시글을 쓰는 부담은 없어요. 단어 하나만 떠올리면 시작이에요.

## 이렇게 써요

**단어를 검색해보세요.** 홈에서 떠오르는 단어를 입력합니다. 띄어쓰기 없이, 한글 10자 또는 영문 20자 이내로 입력해 주세요.

**누군가 먼저 도착했다면 그 방에 들어갑니다.** 다른 사람들의 한마디가 보입니다. 좋아요를 누르거나, 답글을 달거나, 자신의 한마디를 남길 수 있습니다.

**처음 도착한 단어라면 첫 한마디를 남겨보세요.** 그 순간 그 단어의 방이 만들어집니다.

**친구끼리만의 단어도 만들 수 있어요.** 약속한 단어 하나만 공유하면 사실상 그 친구들만의 공간이 됩니다.

## 회원과 비회원

비회원도 단어 방을 보고 댓글을 남길 수 있습니다. 회원이 되면 좋아요, 내 책장(모은 단어들), 알림이 추가로 열립니다.

회원 가입은 구글 계정으로 합니다. 비밀번호는 따로 없어요. 닉네임만 한 번 정해주시면 됩니다.

## 다너의 결

다너는 단어 자체가 주인공인 서비스라 보통 커뮤니티와 결이 조금 다릅니다.

- **추천이 없어요.** 검색해서 도착한 단어만 만나게 됩니다.
- **공개·비공개 설정이 없어요.** 외부 노출이 거의 없어 단어를 공유하는 행위 자체가 입장권이 됩니다.
