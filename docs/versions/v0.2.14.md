## 알려진 잔존

- 모바일 브라우저 일부에서 `background-attachment: fixed`가 무시될 수 있음 (특히 iOS Safari). 그 경우엔 배경이 콘텐츠와 함께 스크롤됨 — 시각적으로는 큰 문제 없어 별도 처리 안 함
- PNG 571KB는 첫 페이지 로드에서 추가 다운로드. cover 한 번 깔리면 캐시되니 둘째 페이지부터는 재요청 없음
