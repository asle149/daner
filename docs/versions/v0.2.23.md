## 사용자 영향

- 관리자가 `/admin` 들어가면 한눈에 보임:
  - 사이트 누적 규모 (회원/단어방/댓글/익명 댓글)
  - 오늘 (KST 자정 기준) 새로 일어난 활동
  - 어떤 단어 방에 사람이 모이는지 (TOP 5)
  - 최근 가입한 닉네임 5명 (ADMIN 표시 포함)
  - 누가 언제 댓글 삭제/단어 방 비우기를 했는지 감사 로그 10건
- 비관리자는 여전히 `/admin` 접근 시 홈으로 자동 redirect

## 알려진 잔존

- 일별 그래프 / 주간 추이는 안 함 — 텍스트 카드로 충분. 필요해지면 v3
- 사용자 목록 페이징 / 검색 X — 5명만 노출
- 감사 로그도 10개 고정. 더 보려면 별도 페이지 필요 (v3)
- 백엔드는 매 호출마다 약 10개 쿼리 — 캐시 없음. 트래픽이 커지면 1분 캐싱 검토
