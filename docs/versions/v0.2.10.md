# v0.2.10

날짜: 2026-05-09
범위: 프론트엔드

## 발견된 이슈

1. 댓글 작성 직후인데 "9시간 전"으로 표시됨. 백엔드 `LocalDateTime`이 timezone 정보 없는 ISO로 직렬화되고, 프론트의 `new Date()`가 이를 로컬(KST)로 해석해 UTC와 KST 차이만큼(9시간) 오차가 누적
2. Composer의 익명 체크박스가 브라우저 기본 파란색이라 다너의 무채색 톤과 안 맞음
3. 단어 방 카운트 "n개의 마음"을 "n개의 이야기"로 변경
4. 익명 체크 상태가 페이지 이동/재접속마다 초기화돼 사용성 떨어짐. 한 번 체크하면 다음에도 같은 상태로 유지되길 원함 (댓글·답글 둘 다)

## 알려진 잔존

- 백엔드는 여전히 timezone-naive `LocalDateTime`. 다른 클라이언트(나중에 모바일 등)가 직접 파싱하면 같은 문제. 근본 fix는 `OffsetDateTime` 또는 Jackson 글로벌 설정. 다음 라운드 후보
- 익명 선호는 이 브라우저(localStorage) 한정. 다른 기기에서 로그인해도 동기화 안 됨. 의도적 — 작은 비밀
