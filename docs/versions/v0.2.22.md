## 발견된 이슈

1. 단어 페이지의 "N개의 이야기" 카운트가 실제 댓글 수와 안 맞음. 예: "페르소나" 가 댓글 1 + 답글 1 = 2 개인데 화면엔 "11개의 이야기" 로 표시. 원인 추정: v0.2.20 이전 코드에서 부모 댓글을 CASCADE 로 삭제할 때 `word.commentCount` 가 답글 수만큼 감소하지 않고 1만 감소 → 누적 mismatch. v0.2.20 부터는 코드 레벨에서 정확히 카운트되어 새 mismatch 는 안 생기지만 옛 누적분 정리 필요
2. 메인 OG 카드가 단색 배경이라 너무 평면적. 사이트 본문의 종이 질감(`background.png`)을 살짝 살렸으면 함
3. 단어 페이지 링크 미리보기에 그 단어 자체가 보이면 좋겠음 — 지금은 모든 페이지가 메인 OG 와 동일

## 알려진 잔존

- OG 이미지에 background.png 를 base64 로 임베드하면 결과 PNG 크기가 살짝 커짐 — 1200×630 에 영향 미미하지만 메신저들이 미리보기 캐싱 갱신할 때 처음 한 번은 느릴 수 있음

- 카운트 mismatch 가 다시 생기지 않게 v0.2.20 의 delete 로직이 `beforeCount - afterCount` 만큼 감소시키도록 해뒀음 — 회귀 모니터링은 필요 없을 듯
