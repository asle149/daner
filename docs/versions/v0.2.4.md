## 사용자 영향

- **첫 방문자 흐름이 명확해짐**: 도메인에 처음 들어오면 `/welcome`에서 4단계 안내를 본 뒤 "시작하기!" 클릭으로 `/`로 진입. 그 이후엔 localStorage 플래그로 다시 안 보임
- **모든 사소한 글씨도 손글씨 폰트 적용** — 메타·시간·라벨까지 톤 통일

- 배포 가이드 문서로 daner.kr 띄우는 단계 한 번에 따라가기 가능

## 알려진 잔존

- TTF 5.4MB 첫 로드 비용은 그대로. 모든 텍스트가 이 폰트를 사용하게 되면서 첫 페인트가 살짝 더 늦을 수 있음 — 필요시 폰트 서브셋팅 또는 `preload` 검토
- 손글씨체가 메타 텍스트(11px 라벨 등)에 적용되면 매우 작은 사이즈에서 가독성 저하 가능 — 사용자 한 바퀴 돌려보고 필요시 일부 메타에만 시스템 폰트 강제
- 첫 방문자 redirect는 클라이언트 JS 의존. SSR 단계에선 여전히 `/`가 렌더되고 즉시 `/welcome`으로 jump. 깜빡임은 최소지만 0은 아님
