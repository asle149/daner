## 사용자 영향
- 손글씨 폰트가 댓글·답글·알림 본문까지 일관되게 적용 → 다너 톤이 화면 전체로 확장
- TTF (5MB)로 되돌아왔으므로 첫 로딩 시 폰트 다운로드량이 약간 늘어남 (`font-display: swap` 덕분에 폰트 로드되기 전에도 시스템 폰트로 즉시 표시됨)
- 홈이 위쪽으로 쏠리지 않고 화면 가운데에 입력창이 자리 — 빈 화면 균형 ↑
- 첫 방문자 환영 인트로는 입력창 위에 자연스럽게 배치

## 알려진 잔존
- TTF가 WOFF2 대비 약 2배 크기 (5MB vs 2.4MB). 이후 다시 WOFF2로 돌릴 의향이 생기면 1줄 변경
- 단어방의 "지금 모이는" 영역이 my-words 행과 같이 떠 있을 때 시각적으로 약간 분리감 — 추후 페이지 하단으로 더 명확히 구분 가능

## 절차 메모
- 본 라운드부터 버전 문서는 두 섹션만 둠: `## 발견된 이슈` + `## TODO`
- 그 외 결과 정리(변경 파일 / 빌드 / 영향 / 잔존)는 라운드 끝나고 회고로 추가
