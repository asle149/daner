- 단어방 하단에서 ? 버튼이 입력칸을 가리지 않음
- 좋아요가 달린 자기 댓글도 "지우기"로 삭제 가능 (관련 좋아요·알림 row가 함께 삭제됨)
- 웰컴 페이지: 첫 문장 굵게+밑줄로 시선 끌기, 군더더기 부수 설명 제거

## 알려진 잔존

- 댓글 삭제 시 `words.comment_count`가 줄지 않음 (기존 코드 그대로). 표시용 카운터라 큰 영향 없으나 차후 라운드에서 정리 예정
- 프로덕션(`daner.kr`)에서 로그인 클릭 시 `localhost:8080`으로 가는 이슈는 코드가 아니라 Vercel 환경변수 누락 때문 (`NEXT_PUBLIC_OAUTH_START_URL` 미설정 → 코드의 localhost 기본값으로 fallback). 본 라운드 범위 밖, 본문 채팅에서 별도 안내
