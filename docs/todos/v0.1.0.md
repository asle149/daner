# v0.1.0 — Backend 기반

날짜: 2026-05-08
범위: 백엔드 단독 (`backend/`)

## 요약

Spring Boot 3.3.5 위에 다너 백엔드 풀 기능 구현. 모든 항목 완료, 109개 테스트 통과.

## 작업 항목별 결과

| 묶음 | 내용 | 핵심 파일 |
|---|---|---|
| 셋업 | Spring Boot + Gradle + 의존성 + 패키지 스캐폴드 + ApiResponse / GlobalExceptionHandler / docker-compose | `build.gradle`, `common/response/`, `common/exception/` |
| 도메인 모델 | 도메인 엔티티 6종 (User/Word/Comment/CommentLike/Notification/AnonymousSession) + Flyway V1 + Repository 통합 테스트 | `*/entity/`, `*/repository/`, `db/migration/V1__init_schema.sql` |
| 단어 정규화 | WordNormalizer 유틸 (trim·NFC·길이·공백 검증) + 단위 테스트 21건 | `common/util/WordNormalizer.java` |
| 인증 | 구글 OAuth + 자체 JWT (access/refresh/signup), `/auth/**` 4종 + 통합 테스트 | `auth/service/`, `common/config/SecurityConfig.java` |
| 단어 도메인 | `GET /words/{word}`, `GET /words/{word}/comments` (cursor + sort) | `word/`, `comment/service/CommentService.listForWord` |
| 글·답글 | 댓글·답글 작성/삭제, 비회원 토큰·라벨, depth-1 답글 제한 | `comment/`, `auth/service/Anonymous*` |
| 좋아요 | 좋아요 토글 (UNIQUE 제약, like_count 증감) | `like/` |
| 홈·인기 | popular_word_daily V3 + @Scheduled 매시간 배치, `/home` | `word/service/PopularWordBatchService`, `HomeService` |
| 프로필 | `/users/me` 책장 (단어별 그룹, cursor) | `user/` |
| 알림 | 이벤트 기반 알림 (`@TransactionalEventListener`, REQUIRES_NEW), `/users/me/notifications` | `notification/` |
| 운영 | Bucket4j rate limit, springdoc Swagger, actuator | `common/ratelimit/`, `application.yml` |
