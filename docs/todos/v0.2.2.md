### 사용자 영향

- **익명으로 작성한 본인 글에 "지우기" 버튼이 정상 노출됩니다.** (`isMine` 플래그 + X-Anonymous-Token 항상 전송)
- 한글 제목들이 모두 Memoment Kkukkuk 손글씨체 + 굵게.
- 홈 my-words 행 색이 한 단계 연해져 입력창에 시선이 더 모입니다.
- 헤더 좌우가 비대칭이었던 게 정돈되고, 도움말은 어느 페이지에서도 우하단에서 호출 가능.
- 책장 책들이 동일한 높이, 폭만 활동량(글 수)에 따라 변화 — 시각적으로 더 정돈됨.
- 폰트 파일 5MB → 2.4MB로 절반 이하. (스키마/배포 영향 없음)

### 알려진 잔존
- Memoment 폰트가 한자나 일부 특수문자엔 글리프가 없을 수 있어 시스템 폰트로 폴백됨 — 정상 동작이지만 사용자가 인지해야 하는 부분.
- `font-weight: 400 700`로 가변 선언했으나 실제 폰트는 단일 weight일 수 있어 브라우저가 synthetic bold(가짜 굵게)로 처리할 가능성. 실제로 너무 두껍거나 가늘면 별도 두께 조정 필요.
