# Daner Todos

> 작업 묶음별 기록. 한 묶음 = 한 파일.

각 파일에는 다듬을 부분 / 발견된 이슈 / 처리 결과를 정리한다.

**규칙**
- 한 묶음의 작업 = 한 todo 파일
- 파일명: `vMAJOR.MINOR.PATCH.md` (semver 느낌, 엄격하진 않음)
- 각 파일 안: 요약 → 다듬을 부분 → 작업 항목 → 변경 파일 → 빌드/테스트 → 영향 → 알려진 이슈
