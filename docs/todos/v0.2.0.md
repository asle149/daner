# v0.2.0 — Frontend MVP

날짜: 2026-05-09
범위: 프론트엔드 (`frontend/`) + 모노레포 재구성

## 요약

Next.js 16 + React 19 + TypeScript + Tailwind 4 + TanStack Query v5 위에 Phase 0–8 풀 기능 구현. 백엔드와 동일 깃 리포(monorepo)로 통합.
