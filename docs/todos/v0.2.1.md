4. dev 표시(N 뱃지·야자수 모양 devtools 버튼) 제거
5. 인기 단어 임계값(하루 10개 이상)
6. 답글 Composer 위치를 답글 아래로
7. 본인 글 삭제 — 익명도, 같은 세션이면 가능하도록
8. 하트 버그 (+2씩 증가, 영원히 증가) + 채운 하트 시각화
9. Memoment Kkukkuk 폰트 통합, DANER 대문자
10. 뒤로가기 화살표 제거
11. 책장 폭을 글 수 기반으로, 살짝 진한 배경 + 윗 모서리 라운드
12. "내 책장" 텍스트 → 아이콘
13. 로그아웃 → /me 페이지로 이동
14. ? 도움말 + 피드백 이메일 (asle14910@gmail.com)

## 변경 사항

### Backend
- `AnonymousLabelService` 라벨에서 숫자 제거
- V4 마이그레이션 — `comments.anonymous_token` (UUID) + 인덱스
- `CommentService.delete`가 `anonymousToken` 인자 추가, 회원 user_id 매칭 OR 비회원 토큰 매칭으로 권한 검증
- `CommentController` DELETE 엔드포인트가 X-Anonymous-Token 헤더 읽음
- `SecurityConfig` DELETE /comments/* permitAll
- `PopularWordBatchService` ≥10 임계값 필터
- `CommentRepository` 정렬에 `c.id` 보조 키 (동률 안정성)

### Frontend
- Header 재구성: ?(좌) / DANER 대문자(중) / 책장 SVG·알림(우) — 뒤로가기 제거, 로그아웃 제거
- HelpDialog (오버레이 + ESC, 피드백 mailto)
- Home 환영 텍스트 (localStorage 게이트), 입력 아래 my-words, "지금 모이는"은 임계값 충족 시
- Composer 위치를 답글 목록 아래로
- ReplyList 답글 사이 점선 구분선
- LikeButton 리팩터: 채운 ♥/빈 ♡, useEffect로 prop sync, mutationFn에 현재 liked 인자 전달
- Bookshelf book width = sqrt(글 수) 매핑 + 배경 hairline/40 + rounded-t-md
- /me 페이지 하단에 로그아웃 버튼
- Memoment Kkukkuk @font-face + .font-display 유틸 (DANER, 단어 제목, 닉네임)
- next.config devIndicators: false, ReactQueryDevtools 제거
